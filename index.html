Live pipeline — one row per quote, joined on Quote No (click a Quote No to open it in CRM). Each bucket shows, per week, the count of quotes reaching that stage and their average completion time; the arrow compares the count to the prior week. Durations: hover for start → end. Drafting = opportunity created → RFQ-completion email (or, if none yet, → Internal/Markup approval raised; or, if none, → Customer approval raised — smaller quotes can skip PR/RFQ) · RFQ Process = PR created → RFQ-completion email · Internal Approval = Quote + Markup Range approval created → modified (when Approved) · Customer Approval = Customer approval created → modified · Work Execution = WO created → booking End Job · Ageing = opportunity created → now. The 3-week trend cards always show the current Mon–Sun calendar week and the two before it, independent of the date filter below.
